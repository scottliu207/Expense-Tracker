## Expense Tracker
### MERN Stack Exercise 

## Usage
### create .env file in the root folder and paste up these

NODE_ENV=production

PORT=5000

MONGO_URI=your_database_uri


## Run script
### in the root folder
### #run backend only
npm start

npm run server(nodemon)

### #run frontend only
npm run client

### #run backend and frontend
npm run dev
